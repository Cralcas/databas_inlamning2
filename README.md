# databas_inlamning2

## Beskrivning
Databas för en bokhandel för att lagra kunder, beställningar, böcker och detaljer om varje beställning.


## Reflektion
För att förbättra sökning i Kund-tabellen används index på email kolumen.
Triggers användes för att uppdatera lager och för att logga när en kund lades till i datbase.
